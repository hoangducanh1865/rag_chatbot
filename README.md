# RAG Chatbot

## Frontend
```
cd frontend
```

To install npm libraries, run:
```
npm install
```

To run frontend, run:
```
npm run dev
```

## Backend
```
cd backend
```

To run backend, run:
```
uvicorn app:app --reload --port 8000
```

## Website
To access the chatbot, go to:
```
localhost:3000
```
